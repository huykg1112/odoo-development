# 📘 KẾ HOẠCH PHÁT TRIỂN ODOO WEBSITE DEVELOPER (CHUYÊN SÂU)

---

## 🎯 1. MỤC TIÊU

Trở thành **Odoo Website Developer chuyên nghiệp**, có khả năng:

* Xây dựng landing page hiện đại trên Odoo
* Phát triển và custom **snippet drag-drop**
* Tích hợp backend + frontend (full flow)
* Tối ưu UI/UX tương đương các website SaaS
* Customize sâu `website` và `website_sale`
    
---

## 🧠 2. YÊU CẦU ĐẦU VÀO

* Đã biết Odoo cơ bản:

  * Model (ORM)
  * View XML
  * Module structure
* Biết cơ bản:

  * Python
  * HTML/CSS/JS

---

## 🗺️ 3. TỔNG QUAN LỘ TRÌNH

| Giai đoạn            | Thời gian | Kết quả đạt được        |
| -------------------- | --------- | ----------------------- |
| QWeb nâng cao        | 3–5 ngày  | Hiểu template engine    |
| Controller + Dynamic | 4–6 ngày  | Page có data            |
| Snippet cơ bản       | 5–7 ngày  | Tạo block kéo thả       |
| Snippet nâng cao     | 1–2 tuần  | Snippet dynamic         |
| Assets + Animation   | 5–7 ngày  | UI hiện đại             |
| OWL framework        | 1–2 tuần  | Component nâng cao      |
| Project thực chiến   | 2–3 tuần  | Landing page hoàn chỉnh |

---

## 🧩 4. CHI TIẾT TỪNG GIAI ĐOẠN

---

### 🔹 GIAI ĐOẠN 1: QWEB MASTER

#### Mục tiêu:

Hiểu cách render UI trong Odoo

#### Nội dung:

* Directive:

  * t-if, t-foreach, t-esc
* Template inheritance
* Layout (`website.layout`)

#### Bài tập:

* Clone page `/about`
* Custom lại layout

---

### 🔹 GIAI ĐOẠN 2: CONTROLLER + DYNAMIC PAGE

#### Mục tiêu:

Hiển thị dữ liệu từ database

#### Nội dung:

* HTTP Controller
* Truyền data vào QWeb
* Routing website

#### Bài tập:

* Page `/students`
* Filter theo category

---

### 🔹 GIAI ĐOẠN 3: SNIPPET CƠ BẢN

#### Mục tiêu:

Tạo block kéo thả trong website builder

#### Nội dung:

* Tạo snippet XML
* Đăng ký snippet
* Hiển thị trong UI builder

#### Bài tập:

* Hero section
* Feature section
* Pricing section

---

### 🔹 GIAI ĐOẠN 4: SNIPPET NÂNG CAO

#### Mục tiêu:

Snippet có logic và dynamic

#### Nội dung:

* Snippet có data từ DB
* Snippet option (edit trực tiếp UI)
* JS option cho snippet

#### Bài tập:

* Snippet hiển thị sản phẩm
* Cho phép:

  * đổi màu
  * đổi title
  * đổi background

---

### 🔹 GIAI ĐOẠN 5: ASSETS + ANIMATION

#### Mục tiêu:

Làm UI đẹp, hiện đại

#### Nội dung:

* Asset bundle (`website.assets_frontend`)
* CSS/SCSS
* Animation (scroll, hover)

#### Bài tập:

* Hero animation
* Scroll effect
* Button hover

---

### 🔹 GIAI ĐOẠN 6: OWL (ADVANCED FRONTEND)

#### Mục tiêu:

Xây dựng component hiện đại

#### Công nghệ:

* OWL

#### Nội dung:

* Component structure
* State management
* Event handling

#### Bài tập:

* Counter component
* Carousel slider
* Testimonial block

---

### 🔹 GIAI ĐOẠN 7: PROJECT THỰC CHIẾN

#### Mục tiêu:

Áp dụng toàn bộ kiến thức

#### Project:

👉 SaaS Landing Page

#### Bao gồm:

##### Sections:

* Hero
* Features
* Pricing
* Testimonials
* CTA

##### Chức năng:

* Snippet reusable
* Dynamic data
* Responsive

---

## 🛠️ 5. CÔNG NGHỆ SỬ DỤNG

* Backend: Python (Odoo ORM)
* Frontend:

  * QWeb
