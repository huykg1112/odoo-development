# SNIPPET "THREE PART TEXT" - QUICK START GUIDE

## ⚡ Khởi Động Nhanh (5 Phút)

### Bước 1: Thêm Snippet Vào Trang
1. Mở website editor (Mode chỉnh sửa)
2. Tìm **"Three Part Text"** trong snippet picker
3. Kéo vào trang bất kỳ vị trí

### Bước 2: Edit Nội Dung

#### Cách 1: Edit Trực Tiếp
```
Kích đôi chuột vào text → Sửa text → Ctrl+S
```

#### Cách 2: Dùng Options Panel
```
1. Click chọn snippet
2. Right side → tìm "Part 1 - Starting Text"
3. Field "Text Content" → nhập text mới
4. Bấm Enter → OK!
```

### Bước 3: Thay Đổi Màu

```
1. Click chọn snippet
2. Right side → "Part 1 - Starting Text"
3. Click "Text Color" → Color picker thoát ra
4. Chọn màu → DONE!
```

### Bước 4: Thay Đổi Kích Thước, Độ Dày, etc.

```
1. Click chọn snippet
2. Right side → "Part 1 - Starting Text"
3. "Font Size" → Chọn size
4. "Font Weight" → Chọn độ dày
5. "Font Style" → Italic or Normal
6. "Text Decoration" → Underline, Strike-through, etc.
```

### Bước 5: Lặp Lại Cho Part 2 và Part 3

```
Part 1 xong → Scroll down → "Part 2 - Middle Text" → Repeat
Part 2 xong → Scroll down → "Part 3 - Ending Text" → Repeat
```

### Bước 6: Cài Đặt Chung (Alignment, Line Height, Padding)

```
1. Scroll down trong options panel
2. "General Settings" section
3. "Text Alignment" → Left, Center, Right, Justify
4. "Line Height" → Chọn khoảng cách dòng
5. "Padding" → Chọn padding xung quanh
```

### Bước 7: Save Trang

```
1. Click button "Save" (phía trên)
2. OK! Snippet được lưu
```

   - Color: Red (#e53e3e)
   - Font Size: 20px
   - Font Weight: Bold (700)
3. Part 2: Edit text → " rằng chúng tôi đã đạt 1 triệu khách hàng"
   - Color: Blue (#3182ce)
   - Font Size: 16px
   - Font Weight: Normal (400)
4. Part 3: Xóa bỏ (để trống)

```
"Nghiên cứu mới cho thấy uống cà phê hàng ngày tăng hiệu suất 30%."
└──────┬────┘ └────────────────────┬───────────────────────────┘
Phần 1          Phần 2 + 3
(Xanh, Bold)    (Đen, Italic, Large)
```

**Các Bước:**
1. Thêm snippet vào trang

   - Color: Green

   - Color: Black
   - Font Style: Italic
   - Font Size: 18px
4. Part 3: " 30%"
   - Color: Red

## 🎯 Tùy Cheinh Nâng Cao

### Thêm TextDecoration (Gạch Chân)

```
1. Click chọn snippet
2. Right side → Part X
3. "Text Decoration" section
4. Click "Underline" → Gạch chân được thêm
5. Hoặc "Strike Through" → Gạch ngang
```

### Thay Đổi Text Alignment

```
1. Scroll down trong options
2. "General Settings"
3. "Text Alignment" → Left, Center, Right, Justify
4. Click → Text được căn chỉnh lại
```

### Thay Đổi Line Height

```
1. Scroll down trong options
2. "General Settings"
3. "Line Height" → 1, 1.4, 1.6, 1.8, 2
4. Click → Khoảng cách dòng thay đổi
```

### Thêm Padding

```
1. Scroll down trong options
2. "General Settings"
3. "Padding" → None, Small, Medium, Large
4. Click → Padding xung quanh được apply
```

## ❓ Câu Hỏi Thường Gặp

**Q: Làm sao để edit text trực tiếp?**
A: Kích đôi chuột vào text → Type → Ctrl+S → Save

**Q: Các tùy chọn ở đâu?**
A: Click chọn snippet → Right side → Options panel

**Q: Có thể xóa Part 2 hoặc 3 không?**
A: Có, nhập text trống (empty string)

**Q: Có thể thêm Part 4, Part 5 không?**
A: Hiện tại chỉ hỗ trợ 3 phần trong snippet default

**Q: Màu không thay đổi?**
A: Check xem Color picker được apply vào đúng part không

**Q: Làm sao để reset về default?**
A: Xóa snippet → Add lại
